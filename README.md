# NPM SCRIPTS

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
# build for production to /dist 
$ npm run build

# watch for changes and deploy it to server (from ./accesses)
$ npm run server-dev
# build for production to /dist and deploy js, css to server (from ./accesses)
$ npm run server-build

# build for production and deploy /dist folder to html.xpager 
$ npm run deploy
```
